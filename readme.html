Installation & Updating Steps

    Please open:
  1. Example 1 : http://localhost/inventory/setup
    Example 2 : https://www.example.com/setup
  2. In that form you will get two options. 1. Install & 2. Update
    • Option 1: Select Install button for new installation.
    • Option 2: Select Update button for updating database.
  3. After Update Process: You need to upload or copy-paste the previous files and folder, Please refere the Help Document. https://pos.creatantech.com/help
    4. Note: Before updating database please take database backup from your server and make it sure.
